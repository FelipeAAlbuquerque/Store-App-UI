# Sticker Store UI

A simple user interface for a Sticker Store application.  
This project provides the front-end UI for browsing, viewing, and purchasing collectible stickers.

## Features

- Browse sticker catalog
- View sticker details
- Add sticker to cart
- Responsive design

## Tech Stack

- React
- HTML/CSS
- JavaScript
